# TT07_power_oneway — 单因素方差分析样本量

## 任务元信息

| 属性 | 值 |
|------|-----|
| **Task ID** | TT07_power_oneway |
| **任务名称** | 单因素方差分析样本量 |
| **任务族** | T - 样本量功效分析 |
| **难度级别** | basic |
| **数据结构** | 无需数据（参数计算） |

## 任务摘要

计算单因素方差分析（One-way ANOVA）所需的样本量。基于组数、效应量（Cohen's f）、显著性水平和检验功效，使用Stata的`power oneway`命令计算每组所需样本量。

## 适用场景

### 场景 1：多组比较研究设计
- 确定比较多个处理组均值差异所需的最小样本量
- 评估现有样本是否足够检测组间差异

### 场景 2：实验设计规划
- 为随机对照试验确定样本量
- 功效分析报告撰写

## 输入与占位符说明

| 占位符 | 含义 | 类型 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `__N_GROUPS__` | 组数 | 整数 | ✓ 是 | 3 |
| `__EFFECT_SIZE__` | 效应量（Cohen's f） | 数值 | ✓ 是 | 0.25 |
| `__ALPHA__` | 显著性水平 | 数值(0-1) | 否 | 0.05 |
| `__POWER__` | 目标检验功效 | 数值(0-1) | 否 | 0.8 |

## 技术说明

- **Stata 版本**：18.0+
- **关键命令**：`power oneway`
- **外部依赖**：无
- **效应量参考**：小效应 f=0.10，中效应 f=0.25，大效应 f=0.40
