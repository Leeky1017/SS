# TT09_power_equivalence — 等价性检验样本量

## 任务元信息

| 属性 | 值 |
|------|-----|
| **Task ID** | TT09_power_equivalence |
| **任务名称** | 等价性检验样本量 |
| **任务族** | T - 样本量功效分析 |
| **难度级别** | basic |
| **数据结构** | 无需数据（参数计算） |

## 任务摘要

计算等价性检验和非劣效检验所需的样本量。基于真实差异、标准差、等价边界、显著性水平和检验功效，使用Stata的`power twomeans, equivalence`命令计算所需样本量。

## 适用场景

### 场景 1：生物等效性研究
- 确定证明两种药物等效所需的样本量
- 仿制药与原研药比较研究设计

### 场景 2：非劣效性临床试验
- 确定证明新疗法不劣于标准疗法所需的样本量
- 功效分析报告撰写

## 输入与占位符说明

| 占位符 | 含义 | 类型 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `__DELTA__` | 真实差异 | 数值 | 否 | 0 |
| `__SD__` | 标准差 | 数值 | ✓ 是 | 1 |
| `__MARGIN__` | 等价边界 | 数值 | ✓ 是 | 0.5 |
| `__ALPHA__` | 显著性水平 | 数值(0-1) | 否 | 0.05 |
| `__POWER__` | 目标检验功效 | 数值(0-1) | 否 | 0.8 |

## 技术说明

- **Stata 版本**：18.0+
- **关键命令**：`power twomeans, equivalence`
- **外部依赖**：无
- **等价边界说明**：通常设为临床意义差异的边界值
