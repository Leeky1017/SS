# TT08_power_logrank — 生存分析样本量

## 任务元信息

| 属性 | 值 |
|------|-----|
| **Task ID** | TT08_power_logrank |
| **任务名称** | 生存分析样本量 |
| **任务族** | T - 样本量功效分析 |
| **难度级别** | basic |
| **数据结构** | 无需数据（参数计算） |

## 任务摘要

计算生存分析中Log-rank检验所需的样本量。基于风险比（Hazard Ratio）、组间分配比例、显著性水平和检验功效，使用Stata的`power logrank`命令计算所需样本量和事件数。

## 适用场景

### 场景 1：临床试验设计
- 确定生存分析研究所需的最小样本量
- 评估检测特定风险比差异所需的事件数

### 场景 2：流行病学研究规划
- 队列研究样本量计算
- 功效分析报告撰写

## 输入与占位符说明

| 占位符 | 含义 | 类型 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `__HR__` | 风险比（Hazard Ratio） | 数值 | ✓ 是 | 0.7 |
| `__P1__` | 组1比例 | 数值(0-1) | 否 | 0.5 |
| `__ALPHA__` | 显著性水平 | 数值(0-1) | 否 | 0.05 |
| `__POWER__` | 目标检验功效 | 数值(0-1) | 否 | 0.8 |

## 技术说明

- **Stata 版本**：18.0+
- **关键命令**：`power logrank`
- **外部依赖**：无
- **风险比说明**：HR<1表示处理组风险降低，HR>1表示风险增加
