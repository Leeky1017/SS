## 任务摘要

使用核方法进行非参数回归分析。`npregress kernel`命令使用核函数加权估计条件均值，支持多个自变量，可处理连续和离散变量的混合情况。

## 适用场景

### 场景 1：非参数条件均值估计
- 不假设函数形式的回归分析
- 多变量非参数回归

### 场景 2：模型设定检验
- 作为参数模型的基准比较
- 检验线性假设的合理性

## 输入与占位符说明

| 占位符 | 含义 | 类型 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `__DEPVAR__` | 因变量 | 变量名 | ✓ 是 | - |
| `__INDEPVARS__` | 自变量列表 | 变量列表 | ✓ 是 | - |
| `__KERNEL__` | 核函数 | 字符串 | 否 | epanechnikov |
