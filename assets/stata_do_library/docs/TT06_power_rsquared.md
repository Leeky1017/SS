# TT06_power_rsquared — 回归R²检验样本量

## 任务元信息

| 属性 | 值 |
|------|-----|
| **Task ID** | TT06_power_rsquared |
| **任务名称** | 回归R²检验样本量 |
| **任务族** | T - 样本量功效分析 |
| **难度级别** | basic |
| **数据结构** | 无需数据（参数计算） |

## 任务摘要

计算回归分析中检验R²显著性所需的样本量。基于目标R²值、预测变量数量、显著性水平和检验功效，使用Stata的`power rsquared`命令计算所需样本量。

## 适用场景

### 场景 1：研究设计阶段
- 确定回归分析所需的最小样本量
- 评估现有样本是否足够检测预期效应

### 场景 2：论文方法学部分
- 为样本量选择提供统计学依据
- 功效分析报告

## 输入与占位符说明

| 占位符 | 含义 | 类型 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `__RSQUARED__` | 目标R²值 | 数值(0-1) | ✓ 是 | 0.1 |
| `__N_PREDICTORS__` | 预测变量数量 | 整数 | ✓ 是 | 3 |
| `__ALPHA__` | 显著性水平 | 数值(0-1) | 否 | 0.05 |
| `__POWER__` | 目标检验功效 | 数值(0-1) | 否 | 0.8 |
