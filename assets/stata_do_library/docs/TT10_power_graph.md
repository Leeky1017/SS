# TT10_power_graph — 功效曲线可视化

## 任务元信息

| 属性 | 值 |
|------|-----|
| **Task ID** | TT10_power_graph |
| **任务名称** | 功效曲线可视化 |
| **任务族** | T - 样本量功效分析 |
| **难度级别** | basic |
| **数据结构** | 无需数据（参数计算） |

## 任务摘要

生成功效曲线图，展示不同效应量下的检验功效变化。使用Stata的`power`命令配合`graph`选项，可视化样本量、效应量与功效之间的关系。

## 适用场景

### 场景 1：研究设计敏感性分析
- 展示不同效应量假设下的功效变化
- 评估样本量选择的稳健性

### 场景 2：论文图表制作
- 生成功效分析可视化图表
- 展示研究设计的统计学依据

## 输入与占位符说明

| 占位符 | 含义 | 类型 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `__TEST_TYPE__` | 检验类型 | 字符串 | 否 | twomeans |
| `__EFFECT_MIN__` | 最小效应量 | 数值 | 否 | 0.1 |
| `__EFFECT_MAX__` | 最大效应量 | 数值 | 否 | 0.8 |
| `__ALPHA__` | 显著性水平 | 数值(0-1) | 否 | 0.05 |
| `__N_SAMPLE__` | 样本量 | 整数 | ✓ 是 | 100 |

## 输出文件清单

| 文件名 | 类型 | 说明 |
|--------|------|------|
| `fig_TT10_power_curve.png` | 图形 | 功效曲线图 |
| `table_TT10_power.csv` | 表格 | 功效值数据 |
| `data_TT10_power.dta` | 数据 | 输出数据 |
| `result.log` | 日志 | 执行日志 |

## 技术说明

- **Stata 版本**：18.0+
- **关键命令**：`power, graph`
- **外部依赖**：无
- **检验类型选项**：twomeans（双样本均值）、oneproportion（单样本比例）等
