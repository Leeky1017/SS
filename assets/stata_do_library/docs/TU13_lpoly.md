# TU13_lpoly — 局部多项式回归

## 任务元信息

| 属性 | 值 |
|------|-----|
| **Task ID** | TU13_lpoly |
| **任务名称** | 局部多项式回归 |
| **任务族** | U - 高级方法 |
| **难度级别** | advanced |
| **数据结构** | 面板/截面数据 |

## 任务摘要

使用局部多项式方法进行非参数回归平滑。`lpoly`命令在每个点使用加权最小二乘法拟合局部多项式，生成平滑的条件均值估计曲线，适用于探索性数据分析和非线性关系可视化。

## 适用场景

### 场景 1：探索性数据分析
- 可视化两变量之间的非线性关系
- 识别数据中的模式和趋势

### 场景 2：非参数估计
- 不假设特定函数形式的回归分析
- 作为参数模型设定的参考

## 输入与占位符说明

| 占位符 | 含义 | 类型 | 必填 | 默认值 |
|--------|------|------|------|--------|
| `__YVAR__` | 因变量 | 变量名 | ✓ 是 | - |
| `__XVAR__` | 自变量 | 变量名 | ✓ 是 | - |
| `__DEGREE__` | 多项式阶数 | 整数 | 否 | 1 |
| `__BANDWIDTH__` | 带宽 | 数值 | 否 | auto |
