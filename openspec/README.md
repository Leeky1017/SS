# OpenSpec

OpenSpec 用于记录可验证的需求增量（spec-first）与运行证据（task runs）。

- Specs：`openspec/specs/**/spec.md`
- Task runs：`openspec/_ops/task_runs/ISSUE-N.md`

## Official layout (CLI)

- Project: `openspec/project.md`
- Handoff: `openspec/AGENTS.md`
- Changes: `openspec/changes/` (archive: `openspec/changes/archive/`)

## Archive

- Archived specs: `openspec/specs/archive/`

## Canonical docs (SS)

- SS constitution (工程圣旨)：`openspec/specs/ss-constitution/spec.md`
- OpenSpec 写作规范：`openspec/specs/openspec-writing-standard/spec.md`
