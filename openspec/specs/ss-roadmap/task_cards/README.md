# SS Task Cards（Issue 蓝图）— 使用说明

本目录已不再存放 task card 文件本体。
task card 已按“主责 spec”分散到：`openspec/specs/<spec-id>/task_cards/`。

重要：task card **不替代** GitHub Issue / Rulebook tasks：
- GitHub Issue：并发与交付入口（任务唯一 ID）
- Rulebook task：执行清单（可勾选）
- Run log：`openspec/_ops/task_runs/ISSUE-N.md`（运行证据账本）

完成态要求（强制）：
- Issue 关闭并 PR 合并后，必须回填该 Issue 对应的 task card：
  - Acceptance checklist 全部打勾（`[x]`）
  - 增加 `## Completion`：PR 链接 + 2-5 条完成要点 + run log 路径
