# UX Task Card: Status, Downloads, and Empty States

## Covers

- 11, 28
- 36
- 62

## Problem (plain language)

状态页是“交付出口”，但现在不够省心：
- 只能逐个下载，缺少“下载全部/打包下载”。
- 空状态只有一行字，没有引导。
- 自动刷新间隔固定且不透明。
- 缺少“分享任务链接”入口（虽然 URL 有 jobId，但用户不知道怎么用）。

## Acceptance

- [ ] 下载支持批量（下载全部或打包）
- [ ] 空状态有引导（告诉用户下一步做什么、可能要等多久）
- [ ] 自动刷新显示间隔，并允许关闭/调整
- [ ] 提供复制/分享当前任务链接入口（包含提示：给同事查看用）

## Dependencies

- 若后端不提供打包下载：前端可先做“批量拉取并客户端打包”的降级方案；或新开后端契约卡。
